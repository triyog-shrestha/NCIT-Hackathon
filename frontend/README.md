CareConnect — BetterHelp-inspired demo

To run locally:
1. cd betterhelp-clone
2. npm install
3. npm start

Notes:
- Front-end only demo built with React (CRA-style). No backend or real authentication included.
- Design changes: different color palette (soft violet), rounded cards, gradient header.
- To add real messaging, booking, and payments, a backend and persistent storage are required.
